# **App Name**: Puff Social Landing Page

## Core Features:

- Location Search: Allow users to search for dispensaries by location.
- Dispensary Display: Display a list of nearby dispensaries with ratings and distance.
- Detailed Reviews: Enable users to view detailed reviews and ratings for each dispensary.

## Style Guidelines:

- Primary color: Forest green (#388E3C) for a natural, relaxed feel.
- Background color: Light green (#E8F5E9), almost white, for a clean and fresh appearance.
- Accent color: Yellow-Green (#AEEA00) for interactive elements.
- Body and headline font: 'PT Sans', a humanist sans-serif.
- Use leaf icons and map markers related to dispensaries.
- Layout: Design a mobile-first layout with intuitive navigation.
- Add subtle transition animations when displaying search results or dispensary details.
